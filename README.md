\# Gesture Rock Paper Scissors 🖐️🎮



Real-time Rock Paper Scissors using \*\*hand gesture recognition\*\*! No buttons - pure hand gestures!



\## 🎯 Features

\- ✅ Live webcam hand tracking (MediaPipe AI)

\- ✅ 3 gestures: ✊ ROCK, ✋ PAPER, ✂️ SCISSORS

\- ✅ Real-time scoring system

\- ✅ 2-second hold to play (prevents spam)

\- ✅ Smooth 30+ FPS gameplay.



\## 🚀 Quick Start



pip install opencv-python mediapipe

python gesture\_rps.py



\*\*Hold each gesture for 2 seconds\*\* → Instant game!
